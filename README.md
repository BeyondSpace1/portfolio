## 🧑‍💻 About

This is my personal portfolio — a dark, editorial-styled single-page application built with vanilla HTML, CSS, and JavaScript. It presents my work at the intersection of **full stack development** and **application security**, backed by a Post-Graduate certification from **IIIT-Bangalore** (in collaboration with NPCI).

> Security isn't a layer on top. It's the foundation.

The portfolio covers my experience in building production-grade systems for finance and healthcare, penetration testing, VAPT, and secure SDLC practices.

---

## ✨ Features

- **Dark editorial design** — terminal-meets-security aesthetic with a mint-green accent system
- **Animated sections** — scroll-triggered fade-up reveals via IntersectionObserver
- **Responsive layout** — fully mobile-optimised with a hamburger nav
- **SPA-like navigation** — smooth scroll with active section highlighting
- **Resume download** — direct PDF access from nav and contact section
- **Live project links** — all 6 projects linked to GitHub or live deployments
- **No frameworks** — pure HTML/CSS/JS, zero build step required

## 📦 Running Locally

No build tools required.

```bash
git clone https://github.com/BeyondSpace1/portfolio.git
cd portfolio
# Open index.html in your browser — that's it.
```

Or use a simple local server for best results:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then open `http://localhost:8000`.
